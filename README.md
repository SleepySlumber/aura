# Fitness Aura

A comprehensive fitness website providing workout plans, nutrition guidance, and health calculators.

## Pages

- **Home** (`index.html`): Welcome page with BMI calculator and quick tips.
- **Workouts** (`workouts.html`): Beginner workout plans and cardio routines.
- **Nutrition** (`nutrition.html`): Calorie calculator and healthy eating tips.
- **Contact** (`contact.html`): Contact form for inquiries.

## Features

- BMI Calculator: Calculate your Body Mass Index and get category.
- Calorie Calculator: Estimate daily calorie needs based on age, gender, and activity level.
- Workout Plans: Beginner-friendly exercises.
- Nutrition Tips: Guidance on healthy eating.
- Responsive Design: Works on desktop and mobile devices.

## Structure

- `index.html`: Home page
- `workouts.html`: Workout plans
- `nutrition.html`: Nutrition guide
- `contact.html`: Contact page
- `css/style.css`: Stylesheet with modern design
- `js/script.js`: JavaScript for calculators and interactivity
- `images/`: Directory for images (currently empty)

## How to Run

To view the website locally, start a simple HTTP server:

For Python 3:
```
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

For Node.js (if installed):
```
npx http-server
```
